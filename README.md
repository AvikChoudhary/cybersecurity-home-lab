# Cybersecurity Home Lab - Virtual Penetration Testing Environment

Built while completing Zero To Mastery Ethical Hacking Bootcamp & Andrew Ramdayal CompTIA Security+ course.

## Lab Setup
- Attacker: Kali Linux
- Target: Metasploitable 2, Windows 7, Windows 10
- Hypervisor: VirtualBox

## Step-by-Step Instructions
1. Install VirtualBox
2. Download Kali Linux + Metasploitable 2 + Windows 7, 10 ISO
3. Create VMs in Internal Network mode
4. From Kali run commands

## Tools Practiced
Nmap, Msfconsole, Msfvenom, Whatweb, Sherlock, TheHarvester, Spyder, VirtualBox
Nessus, Payload testing and many other tools

## Advanced Lab Work (Windows 7 & Windows 10 Targets)

- Set up vulnerable Windows 7 and Windows 10 VMs as targets
- Performed service enumeration with Nmap
- Used msfconsole to exploit known vulnerabilities (e.g., EternalBlue on Windows 7)
- Generated payloads with msfvenom and demonstrated Meterpreter session


## What I Learned
Safe lab creation, reconnaissance phase, scanning, exploitation, post-exploitation, and reporting, legal & ethical boundaries.
